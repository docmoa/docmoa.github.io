# Alibaba CLI 설정

# 1. CLI 설치

# 1.1 Download 방식

  • Install guide : https://partners-intl.aliyun.com/help/doc-detail/139508.htm (opens new window)
  • Release Download Page : https://github.com/aliyun/aliyun-cli/releases (opens new window)
    • CLI 릴리즈 페이지에서 OS에 맞는 파일을 다운로드하여 사용

# 1.2 MAC - brew 설치

  • Install guide : https://formulae.brew.sh/formula/aliyun-cli (opens new window)
    brew install aliyun-cli
    1

# 2. 구성 설정

인증 메커니즘이 4가지이며 구성 설정시 --mode 에 Credential type을 넣어서 구성하게 됨

Credential type Limit Interactive credential configuration (fast) Non-interactive credential configuration
AK Use an AccessKey ID and an AccessKey secret to authorize access. Configure AccessKey credential (opens new window) Configure AccessKey credential (opens new window)
StsToken Use a Security Token Service (STS) token to authorize access. Configure STS token credential (opens new window) Configure STS token credential (opens new window)
RamRoleArn Use the role of a Resource Access Management (RAM) user to authorize access. Configure RamRoleArn credential (opens new window) Configure RamRoleArn credential (opens new window)
EcsRamRole Use the RAM role of an Elastic Compute Service (ECS) instance to authorize password-free access. Configure EcsRamRole credential (opens new window) Configure EcsRamRole credential (opens new window)

# 2.1 AK 생성

AccessKey 방식의 인증 정보가 없는 경우 아래와 같이 생성

  • Alibaba Cloud 홈페이지의 우측 상단(변경될 수 있음) 클릭하여 Accesskey Management 클릭
  • AccessKey Pair 항목에서 Create AccessKey Pair 를 클릭하여 AK(AccessKey)를 신규로 생성

# 2.2 CLI 구성

  • --mode 에는 Credential Type을 지정
